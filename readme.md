# Todo! Mobile Project

This is an individual project in order to improve my learning

## ⚙️ Installation

Install Dependencies

```bash
  npm install
```

Run Project

```bash
  npm start
```

## 🖥️ Technologies Used

- React (18.0.14)
- React-Native (0.71.3)
- Typescript (4.9.3)

## 📚 Libraries Used

- Expo
- Axios
- React-Native-Bouncy-Checkbox
- Phosphor React Native
- Styled Components

## 🚀 Features

- Change of State
- Use Styled Components for styles
- Connection with API
